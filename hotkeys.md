# HOTKEYS 
  A map for hotkeys to remember them more well and use them more often

### Tools
 - `V` Select
 - `A` Dircet Select
 - `M` Rectangle
 - `L` Elipse
 - `T` Text
 - `shift + O` Artboard
 - `shift M` Merge shapes

### Functions
#### Common usage:
 - `ctrl + [` | `Ctrl + ]` Change Z index
 - `ctrl + 2` | `ctrl + shift + 2` Lock | Unlock selection
#### Specific usage:
 - `ctrl + shift + o` Transform text into shapes 

### Panels
 - `F6` Colors
 - `F7` Layers
 - `ctrl + F9` Gradients
 - `???` Transform
 - `shift + F7` Align
 - `???` Pathfinder
